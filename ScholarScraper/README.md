# Scholar Scraper - Google 学术助手

一个专为研究人员设计的 Chrome 插件，旨在自动化抓取、追踪和分析 Google 学术（Google Scholar）的引用数据，采用现代化的 Material Design 风格设计。

## 核心功能

### 1. 学术看板（Profile Tracking）
- **指标监控**：将常用的学者主页（包括自己）保存为卡片，在侧边栏实时查看**总引用次数 (Citations)** 和 **h-index**。
- **自动巡检**：打开插件时自动触发 "Refresh All"，批量更新所有收藏学者的最新数据，无需手动逐个点击。
- **状态可视化**：清晰展示上次更新时间，并通过颜色区分引用数（红色）和 h-index（蓝色）。

### 2. 第一作者（一作）追踪
- **智能筛选**：设置你的姓名后，插件会在后台自动分析学者的论文列表，精准筛选出你是**第一作者**的论文。
- **下拉展示**：无需跳转页面，在卡片上直接点击 "Papers" 按钮，即可展开查看一作论文详情（标题、年份、引用数）。
- **模糊匹配**：支持全名（如 `Ziqi Wang`）及首字母缩写（如 `Z Wang`）的智能匹配逻辑。

### 3. 极速交互
- **一键跳转**：点击卡片任意空白处，即可让主窗口直接跳转到对应学者的 Google Scholar 主页。
- **无感更新**：采用静默标签页技术，在后台完成数据抓取，不干扰当前浏览体验。

## 安装步骤

1. 下载或克隆本仓库。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`（扩展程序管理）。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目的 `ScholarScraper` 文件夹。

## 使用指南

### 基础设置
1. **设置自己**：在插件顶部的 "Author Name" 输入框填写你的姓名拼写（如 `Ziqi Wang`），这将用于全局的一作识别。
2. **添加学者**：在 "Profile URL or ID" 输入框粘贴 Google Scholar 个人主页链接或 ID，点击 **Add**。

### 日常使用
- **查看更新**：每次打开侧边栏，插件会自动刷新所有数据。
- **查看一作**：点击卡片右侧的 **Papers** 胶囊按钮，展开查看该学者名下属于你的一作论文。
- **访问主页**：点击卡片主体，浏览器将跳转至该学者的 Google Scholar 详情页。
- **删除卡片**：点击卡片右上角的 **×** 号即可移除关注。

## 注意事项

- **反爬策略**：插件内置了随机延迟逻辑以规避检测，但如果收藏人数过多（>20人），频繁刷新仍可能触发 Google 的验证码（CAPTCHA）。
- **异常处理**：如果卡片显示 "CAPTCHA!" 或 "404"，请点击卡片跳转到页面手动通过验证或检查链接有效性。
