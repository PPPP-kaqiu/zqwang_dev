# 字节跳动简历自动填写助手 (AutoResume Filler)

这是一个 Chrome 浏览器扩展，采用了现代化的**侧边栏 (Side Panel)** 设计，旨在帮助你在字节跳动校园招聘网站上自动填写简历信息。

## 功能特点

- **侧边栏设计**：在浏览器右侧常驻显示，方便一边浏览网页一边管理数据。
- **可视化表单**：告别繁琐的 JSON 编辑，提供直观的输入框界面。
- **智能匹配**：采用基于评分的匹配算法，即使网页标签略有不同也能精准识别。
- **数据隐私**：所有数据仅保存在你的本地浏览器中，不上传任何服务器。

## 安装步骤

1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
2. 打开右上角的开关，启用 "开发者模式" (Developer mode)。
3. 点击左上角的 "加载已解压的扩展程序" (Load unpacked) 按钮。
4. 在弹出的文件选择框中，选择本项目中的 `AutoResume` 文件夹。

## 使用方法

1. **打开侧边栏**：
   - 点击浏览器右上角工具栏中的 **AutoResume 图标**。
   - 浏览器的右侧会自动展开扩展程序的侧边栏界面。

2. **配置简历信息**：
   - 在侧边栏的表单中，依次填写 **基本信息**、**教育经历** 和 **工作/实习经历**。
   - 填写完毕后，点击底部的 **"保存信息"** 按钮。

3. **自动填写**：
   - 打开或刷新字节跳动校园招聘简历编辑页面：`https://jobs.bytedance.com/campus/resume/edit`。
   - 再次点击侧边栏底部的 **"自动填写"** 按钮。
   - 插件将自动扫描页面并填入你保存的信息。

## 常见问题

- **点击图标没反应？**：请确保已刷新浏览器，并且是在常规网页上点击（某些 Chrome 系统页面可能不支持侧边栏）。
- **字段未填充？**：插件使用智能模糊匹配，但如果网页结构发生剧烈变化，可能需要更新匹配规则。
- **如何重置？**：点击侧边栏右上角的“重置”按钮即可恢复默认示例数据。
