# 自动简历填写助手 (AutoResume Filler)

这是一个基于 AI 驱动的 Chrome 浏览器扩展，旨在帮助你一键填写各大招聘网站的简历表单。它结合了传统的正则匹配和先进的 AI 大模型技术，能够适应各种复杂的前端框架（React/Vue）和表单结构。

## ✨ 核心功能

*   **侧边栏 (Side Panel)**：常驻浏览器右侧，不会遮挡页面内容，一边浏览一边操作。
*   **🤖 AI 智能填写**：
    *   **语义理解**：自动识别“姓名”、“Name”、“Your Full Name”等不同写法。
    *   **上下文感知**：智能分析输入框周围的文字，准确判断字段含义。
    *   **框架突破**：专门针对 React、Vue 等现代前端框架优化，解决“填了但没反应”的顽疾。
    *   **流式反馈**：像黑客帝国一样实时显示 AI 的思考和填写过程。
*   **⚡ 正则急速填写**：对于标准表单，毫秒级完成填写，无需等待 AI。
*   **通用性强**：支持文本框、多行文本域 (Textarea)、下拉菜单 (Select) 以及富文本编辑器 (ContentEditable)。

## 🚀 安装与使用

### 1. 安装插件
1.  下载本项目代码。
2.  打开 Chrome 浏览器，访问 `chrome://extensions/`。
3.  开启右上角的 **"开发者模式" (Developer mode)**。
4.  点击 **"加载已解压的扩展程序"**，选择 `AutoResume` 文件夹。

### 2. 配置简历
1.  点击浏览器工具栏的 **AutoResume 图标** 打开侧边栏。
2.  在表单中填入你的简历信息（姓名、学历、工作经历等）。
3.  点击底部的 **"保存信息"**。

### 3. 配置 AI (推荐)
1.  点击侧边栏顶部的 **⚙️ (设置)** 图标。
2.  输入你的 OpenAI 格式 API Key（支持 DeepSeek、OpenAI 等）。
    *   *默认已预置了一个 DeepSeek 测试 Key。*
3.  点击 **"保存设置"**。

### 4. 一键填写
*   打开任意招聘网站（如字节跳动校招官网）。
*   点击 **"✨ AI 填写"**：插件会自动扫描页面结构，思考并填入内容。
*   点击 **"正则填写"**：尝试使用本地规则快速填写。

## 🛠️ 常见问题

**Q: 点击按钮没反应？**
A: 请尝试刷新网页。新安装插件后，旧的标签页需要刷新才能加载插件脚本。

**Q: AI 显示填写完成，但网页没变化？**
A: 插件使用了模拟用户输入的底层技术，通常能触发网页的保存机制。如果仍有问题，请尝试手动点击一下任意输入框，看是否会触发页面的验证逻辑。

**Q: 支持哪些网站？**
A: 理论上支持所有基于 HTML DOM 的网站。对于使用 Canvas 绘制的非标准界面暂不支持。

## 📦 技术栈
*   Manifest V3

*   Vanilla JS (无框架依赖，轻量极速)
